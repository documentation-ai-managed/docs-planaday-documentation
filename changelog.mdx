## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added advanced reporting dashboard with customizable metrics
- Introduced bulk student import functionality with CSV validation
- Implemented real-time course progress notifications
- Added support for course prerequisites and learning paths

## Improvements

- Enhanced API rate limits for enterprise customers
- Improved search functionality across all resources
- Updated user interface with modern design elements
- Optimized database queries for faster loading times

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched comprehensive REST API for full system integration
- Added OAuth 2.0 authentication support
- Introduced webhooks for real-time event notifications
- Implemented advanced permission system for granular access control

## Breaking Changes

- Updated API endpoints to use versioned paths (/v2/)
- Changed authentication flow to require API keys or OAuth tokens
- Modified response formats for better consistency

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Added course completion certificates with customizable templates
- Introduced student progress tracking with detailed analytics
- Implemented automated email notifications for course milestones
- Added support for multiple instructors per course

## Bug Fixes

- Fixed issue with student enrollment dates not saving correctly
- Resolved problem with bulk email sending failures

- Fixed compatibility issues with older web browsers
